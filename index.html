Project:

"PHP/Mysql project with self-developed framework"

SportBuddy

Specs and goals

I developed this application without a PHP framework. My goal was to fully understand every aspect of an MVC (Model-View-Controller), so I got a lot of experience. The application is working and the idea was to create a webapp to help people to find a buddy if they want to do some sport, but they are alone. I've built in authorization and authentication, a normal user can register, login, create an event, update own userdata, update/delete own events, join/leave or make a comment on an event. For admin there is extra space to change everyone's details or even see events already expired.

Tools and methods used

Front-end: HTML&CSS, Javascript / JQuery, AJAX, Bootstrap
Back-end: PHP 7.1, MySQL, XAMPP, phpMyAdmin, MySQL Workbench
Tools: Google Maps API, Composer autoload, SimpleRouter (skipperbent), SendGrid, own session class, own validation class

Progress

The app is working, if you try it and register, you will receive a confirmation email (might be in spam folder), you need to click on a link and you are free to login. This email is under construction, I need to add a possibility to unsubscribe etc... The latest updates are some AJAX calls (checks if email address already exists) and a Google Maps API integration. If you create a new event and you start to write a location, it gives you instant address recommendations from which you have to choose. Then your event will be shown on a map.

Project:

"TodoList API"

TodoList API

Specs and goals

The idea is to create an API with full CRUD functionality. I did this through a simple todo list, you can create a todo item with title, description, category and a date. Connected with a simple HTML/JS file you can get the data and create new todo item. The API is deployed on: https://gregstodoapi.herokuapp.com/todo
You can test it HERE
The goal is to be fully functional with authentication, which is being developed now.

Tools and methods used

The API runs on Laravel 5.7.3, the database is MySQL, the application connects to the API with AJAX and uses JSON data
